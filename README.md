# RANDON-PAGES

Repositorio para publicar páginas HTML estáticas con GitHub Pages.

## Uso rápido

1. Sube tus archivos `.html` al repositorio (en la raíz o carpetas).
2. Ve a **Settings > Pages**.
3. En **Build and deployment**, elige:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (o la rama que uses), carpeta `/ (root)`
4. Guarda y abre la URL publicada por GitHub Pages.

## Archivos de ejemplo

- `/index.html`: portada principal.
- `/paginas/ejemplo.html`: ejemplo de página estática adicional.
